VidConnect is a free Video Calling & Chat application built using ZegoCloud's free UiKit and Api Keys.
It is build using React , Framer Motion , React Icons.
It has enhanced features like :
Realtime video calling & chat with multiple users.
Dark & Light Mode for enhancing user experience.
Beautiful Home & About Page for user experience.
Create a room for video call and share anywhere.
Enhanced feature like screen sharing in video call.
Great Quality of Call.

Project is deployed on vercel do check it out !
